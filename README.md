# -Naver-Boostcourse-Android-Programming---final

[네이버 부스트코스] 안드로이드 과정

부스트코스 안드로이드 과정에서 제작한 안드로이드 어플리케이션.

## Features
- API 서버로부터 가져온 영화목록, 영화상세 등의 정보를 표시
- 영화에 대한 좋아요/싫어요 기능과 평점 및 한줄평 작성 기능
- 이미지 확대/축소 및 이동이 가능한 사진보기 기능과 유튜브 API를 활용한 영상재생 기능
